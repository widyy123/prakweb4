Penjelasan kode:
Saya membuat kode ini untuk menentukan zodiak berdasarkan tanggal dan bulan lahir yang diinput oleh pengguna sesuai dengan tugas yang diberikan. untuk cek nya melalui terminal dengan menuliskan node index.js. Di awal, saya pakai readline untuk program  menerima input. kemudian saya membuat interface yang berfungsi untuk menghubungkan proses input dan output di terminal. pada program ini kemudian akan meminta pengguna memasukkan tanggal dan bulan lahir. Data yang saya gunakan itu berbentuk angka. Setelah itu saya membuat variabel untuk menyimpan hasil zodiak. Bagian utama dari kode ini adalah percabangan menggunakan if-else. Di bagian ini proses penentuan zodiak dilakukan dengan mencocokkan tanggal dan bulan yang sudah ditentukan untuk setiap zodiak. Saya juga menambahkan informasi untuk memastikan bahwa tanggal dan bulan yang dimasukkan benar. Jika tidak sesuai, maka program akan menampilkan pesan bahwa input nya itu tidak valid.
Program menampilkan hasil berupa tanggal, bulan, dan zodiak yang sesuai, kemudian proses input ditutup agar program selesai dengan baik. Output dari kode ini meminta pengguna memasukkan tanggal dan bulan lahir, lalu menampilkan hasil zodiak berdasarkan data yang dimasukkan.
